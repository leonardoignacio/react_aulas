O que é React?

React é uma biblioteca JavaScript para construir interfaces de usuário. Ele permite criar UIs complexas a partir de pequenas peças de código isoladas chamadas "componentes".

UI Declarativa

Você descreve como sua UI deve se parecer, e o React cuida de atualizar o DOM de forma otimizada quando os dados mudam. Você diz *o que* quer exibir, não *como* fazer.

Componentes e JSX

Componentes são blocos de construção reutilizáveis que encapsulam sua própria lógica e aparência. JSX é uma extensão de sintaxe que permite escrever marcação semelhante a HTML no JavaScript.

State e Props

São os dois tipos de dados que controlam um componente.

Props (Propriedades)

Passam dados de um componente pai para um filho. São somente leitura.

State (Estado)

Dados gerenciados internamente por um componente. Quando muda, o componente renderiza novamente.

O DOM Virtual

É uma cópia do DOM real mantida em memória, que torna o React rápido.

Processo de Atualização

Quando o estado muda, o React cria um novo DOM Virtual, compara com o antigo (diffing), e atualiza no DOM real apenas o que mudou, de forma otimizada.
