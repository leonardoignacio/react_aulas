Módulo 1: Conectando o Front-end ao Mundo

Aplicações web modernas raramente vivem isoladas; elas precisam buscar, exibir e enviar dados. Este módulo introduz o conceito de APIs, como nos comunicamos com elas usando AJAX, e as ferramentas que tornam esse processo simples no React: Axios para requisições e `json-server` para simular um backend durante o desenvolvimento.

O que é AJAX e uma API?

**API (Application Programming Interface)** é um conjunto de regras que permite que diferentes sistemas de software se comuniquem. Em nosso caso, é a "ponte" entre nosso app React (front-end) e um servidor que guarda os dados (back-end). **AJAX (Asynchronous JavaScript and XML)** é a técnica que permite ao nosso app fazer requisições a essa API em segundo plano, sem precisar recarregar a página inteira.

Axios: Seu Canivete Suíço para Requisições

Axios é uma biblioteca JavaScript que simplifica o processo de fazer requisições HTTP. Ela é baseada em Promises, tornando o código assíncrono mais limpo e legível. A operação mais comum que faremos é a `GET`, para buscar dados de um servidor.

import axios from 'axios';

// Exemplo de como buscar dados
axios.get('https://api.exemplo.com/dados')
  .then(response => {
    // Sucesso! Os dados estão em response.data
    console.log(response.data);
  })
  .catch(error => {
    // Ocorreu um erro na requisição
    console.error('Erro ao buscar dados:', error);
  });

Simulando uma API com `json-server`

Enquanto o back-end está sendo desenvolvido, não podemos parar. `json-server` é uma ferramenta incrível que cria uma API REST completa a partir de um simples arquivo .json. Isso nos permite construir e testar nosso front-end como se estivéssemos conectados a uma API real.
