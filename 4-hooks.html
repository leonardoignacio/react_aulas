Módulo 1: O Poder dos Hooks

Hooks são funções que permitem que você "engate" (hook into) o estado e o ciclo de vida do React a partir de componentes de função. Eles revolucionaram a forma como escrevemos componentes, tornando o código mais simples, reutilizável e fácil de entender. Vamos conhecer os principais.

useState

O Hook mais fundamental. Ele adiciona estado a um componente de função, permitindo que ele guarde informações que mudam ao longo do tempo e reaja a essas mudanças.

const [estado, setEstado] = useState(valorInicial);

useEffect

Permite executar "efeitos colaterais" em seus componentes. Isso inclui buscar dados de uma API, manipular o DOM diretamente ou salvar dados no `localStorage`.

useEffect(() => {\n  // Código do efeito\n}, [dependencias]);

useContext

Permite acessar um valor de um "Contexto" do React sem precisar passar props por múltiplos níveis de componentes (evitando "prop drilling").

useRef

Fornece uma maneira de acessar nós do DOM diretamente ou de manter uma variável mutável que não causará uma nova renderização quando for alterada.
